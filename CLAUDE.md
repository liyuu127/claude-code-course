# Claude Code Course 项目

## 项目简介

这是一个用于学习和演示 Claude Code 功能的课程项目。

## 开发规范

### 代码风格
- 使用清晰的变量和函数命名
- 添加必要的注释说明复杂逻辑
- 遵循各语言的最佳实践

### Git 提交规范
- 使用语义化提交信息
- 格式：`<type>: <description>`
- 类型：feat, fix, docs, style, refactor, test, chore

## Claude 工作偏好

- 在进行重大代码修改前，先与用户确认方案
- 优先阅读现有代码，理解上下文后再修改
- 保持代码简洁，避免过度设计
